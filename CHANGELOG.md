# CHANGELOG

## 多账号版

- 增加 `accounts.json` 多账号配置。
- 第一次安装只录入 1 个账号。
- `cdt` 菜单新增“多账号管理”：查看、增加、删除、编辑、启用、禁用账号。
- cron 每分钟遍历所有启用账号。
- 每个账号独立阈值、日报时间、流量熔断通知、月初自动开机通知。
- Telegram `/status`、`/traffic`、`/bill`、`/report` 支持所有账号汇总。
- Telegram `/startvps 账号名或序号`、`/stopvps 账号名或序号` 支持指定账号。
- 继续保留：北京时间日报、流量熔断通知只发一次、月初 00:05 后检测自动开机。

## 菜单结构优化

- `cdt` 进入后先显示所有账号列表。
- 输入账号序号后进入单账号控制菜单，可单独查看状态、发送日报、开机、关机、执行检查、编辑、启用、禁用、删除。
- 主菜单保留增加账号、删除账号、全局设置、完全卸载程序。
- 全局设置中集中管理 TG Bot 服务、全局保活、日志、crontab、通知状态和配置文件。
